# Project is bootstrapped with create-next-app

Command for starting bootstrapping

```bash
npx create-next-app@latest nem-lager --ts --tailwind --eslint --app --src-dir --turbo --use-pnpm
```

## Init Shadcn UI and darkmode

```bash
# init Shadcn
pnpm dlx shadcn@latest init
```

And follow this [guide](https://ui.shadcn.com/docs/dark-mode/next) for implementing theme switching.

## Init Drizzle ORM with LibSQL

Follow this [guide](https://orm.drizzle.team/learn/tutorials/drizzle-with-turso) in order to setup a database.

## Init Lucia-Auth with Drizzle ORM

Follow this [guide](https://lucia-auth.com/database/drizzle) in order to setup Lucia with Drizzle, and follow this [guide](https://lucia-auth.com/tutorials/username-and-password/nextjs-app) to setup authentication with email and password.
